## Install Theme

Move `nikola-onedark` to themes

```sh
sudo mv nikola-onedark /usr/share/themes
```

Now edit `~/.config/gtk-3.0/settings.ini` and add this:

```sh
# ~/.config/gtk-3.0/settings.ini
gtk-theme-name=nikola-onedark
```
